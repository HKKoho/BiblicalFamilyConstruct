# Biblical Family Construction

> "Happy families are all alike; every unhappy family is unhappy in its own way." - Leo Tolstoy

## About

Biblical Family Construction is an interactive web application designed to help believers build God-honoring families through biblical wisdom and guidance. Based on the book exploring God's teachings on various aspects of family life, this app covers:

- Marriage foundations and covenant love
- Communication and conflict resolution
- Parenting and spiritual leadership
- Intimacy, unity, and family roles
- Financial stewardship and extended family relationships
- Life transitions and divine calling

## 10 Chapters

This application is based on a 10-chapter study:

1. **Sacrificial Love** - Living out true love through self-sacrifice (1 Corinthians 13)
2. **Pre-Marriage** - God's design for relationships before marriage (Genesis 1:27-28, 2:18-25)
3. **Covenant Love** - Building marriage on God's covenant (Song of Songs 3:1-11)
4. **Husband & Wife** - Biblical roles in marriage (Ephesians 5:22-33)
5. **Marital Intimacy** - Cultivating healthy physical intimacy (1 Corinthians 7:1-9)
6. **Complementarity** - How spouses complement each other (1 Peter 3:1-9)
7. **Family Harmony** - Creating peace and unity (Genesis 50:15-21)
8. **Sexual Purity** - Maintaining faithfulness (Proverbs 5:1-14, Matthew 5:27-30)
9. **Contentment** - Finding satisfaction in God's provision (1 Samuel 1:1-11)
10. **Divine Calling** - Understanding marriage and singleness (Matthew 19:3-12, 1 Corinthians 7:25-35)

## Features

- **Bilingual Support**: Available in English and Traditional Chinese (繁體中文)
- **Interactive Guidance**: AI-powered biblical advice using Gemini API
- **10 Core Topics**: Covering all aspects of biblical family construction
- **Resources**: Book extracts and audio testimonies
- **Beautiful UI**: Modern, responsive design with warm, welcoming aesthetics

## Run Locally

**Prerequisites:** Node.js

1. Install dependencies:
   ```bash
   npm install
   ```

2. Set the `GEMINI_API_KEY` in `.env.local`:
   ```
   GEMINI_API_KEY=your_api_key_here
   ```

3. Run the app:
   ```bash
   npm run dev
   ```

## Technology Stack

- React 19 with TypeScript
- Vite for build tooling
- Tailwind CSS for styling
- Google Gemini API for AI-powered guidance
- Heroicons for icons

## Purpose

This application aims to help believers develop proper concepts about family and marriage relationships, understanding them comprehensively through biblical teachings, and establishing family life that aligns with God's will to be a beautiful witness for Christ.
